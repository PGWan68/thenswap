# BC DEX - 去中心化交易所

一个类似于Uniswap的简易DEX交易平台，支持钱包连接、代币管理和基础交易功能。

## 功能特性

- **钱包连接**：支持连接MetaMask等Web3钱包
- **资产管理**：查看钱包中已有的代币余额
- **可交易币种**：展示所有可交易的代币列表
- **代币兑换**：实现基础的代币兑换功能
- **响应式设计**：适配不同屏幕尺寸

## 快速开始

### 方法一：直接使用HTML文件

1. 在浏览器中打开 `dex.html` 文件
2. 点击"连接钱包"按钮
3. 开始使用DEX平台

### 方法二：使用React项目（需要安装依赖）

1. 安装依赖：
   ```bash
   npm install
   ```

2. 启动开发服务器：
   ```bash
   npm run dev
   ```

3. 在浏览器中打开 http://localhost:5173

## 技术栈

- **React**：前端框架
- **Web3.js/Ethers.js**：区块链交互
- **CSS3**：样式设计
- **Vite**：构建工具

## 项目结构

```
├── src/
│   ├── App.jsx          # 主应用组件
│   ├── main.jsx         # 应用入口
│   └── index.css        # 全局样式
├── index.html           # 项目入口HTML
├── vite.config.js       # Vite配置
├── package.json         # 项目依赖
└── dex.html             # 独立HTML版本
```

## 注意事项

- 当前版本使用模拟数据，实际使用时需要连接到真实的区块链网络
- 交易功能仅为演示，实际交易需要与智能合约交互
- 建议使用MetaMask等钱包进行测试
